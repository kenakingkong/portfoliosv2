# postfoliosv2

## stack
- NuxtJS & Vue
- Hosted on Vercel
- Data in AWS & Firebase Realtime Database (to be moved)

## setup

**install deps**

```bash
pnpm install
```

**start dev server**

```bash

# pnpm
pnpm run dev

```

**build prod app**

```bash

# pnpm
pnpm run build

```

**preview prod app**

```bash

# pnpm
pnpm run preview

```
